# 72-Hour Breach Notification Checklist

- [ ] Time of awareness recorded.
- [ ] DPO notified.
- [ ] Incident severity assessed.
- [ ] Affected data categories identified.
- [ ] Number of affected data subjects estimated.
- [ ] Containment actions completed.
- [ ] NPC notification filed (if required) before 72 hours.
- [ ] Data subjects informed (if required).
- [ ] Final incident report archived.
